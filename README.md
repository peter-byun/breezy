# Breezy

A delightful cross-platform flash card app.  
Backend is a private repo for security reasons.

## Tech Stack

- Next.js
- UI
  - [Radix](https://www.radix-ui.com/)
  - modals: [useOverlay](https://www.slash.page/libraries/react/use-overlay/src/useOverlay.i18n)
- Style
  - emotion
- Interactions
  - react-spring
  - use-gesture
- Network
  - axios
  - react-query
- Utility
  - [remeda](https://remedajs.com/)
  - [usehooks-ts](https://usehooks-ts.com/introduction)
  - [Suspensive](https://suspensive.org/en/docs/react/Suspense)
- State Management
  - Try flattening components using the composition pattern
  - Query parameters
  - For dynamic regional states, use [Jotai](https://jotai.org/)
